РАЗВЕДКА РЫНКА

⏱ 15 МИН
ШАГ 1 • PERPLEXITY

Наши лучшие друзья — URL конкурентов и NotebookLM со встроенным поиском. AI забирает из интернета актуальные конкурентные преимущества. Это язык эксклюзивных преимуществ.

⚠ Perplexity работает в России без VPN. Регистрация через email — 2 минуты.
  • Откройте perplexity.ai — New Thread
  • Вставьте промпт №1, заменив [НИША] и [ГОРОД]
  • Из ответа скопируйте только URL-ссылки — сохраните в Google Docs
  • Вставьте промпт №2 — скопируйте весь текст ответа целиком — пойдёт в NotebookLM
ПРОМПТ №1 – КОНКУРЕНТЫ
Подбери 10 сайтов по нише [НИША] в городе [ГОРОД] с высокими рейтингами на Google и Яндексе. Для каждого сайта дай: – Название компании – Клиентский сегмент (кто покупает) – Основной оффер (что обещают) – Ссылку URL главной страницы – Каналы трафика, если видно (соцсети, контекст, SEO) – Чем они отличаются от других, в чем их фишка
ПРОМПТ №2 – БОЛИ АУДИТОРИИ
У тебя есть реальные боли, страхи и возражения клиентов по нише [НИША], которые они пишут в [ФОРУМЫ / ОТЗЫВЫ / ЧАТЫ]. Структурируй эти данные по категориям: – Страхи до покупки / услуги – Ожидания от результата – Основные возражения против покупки – Ошибки, которые клиенты уже совершили – Как они выбирают исполнителя – На что больше всего обращают внимание по цене Сделай список из 20–30 самых частых и сильных формулировок, сохраняя язык клиентов.
✓ ЧТО ПОЛУЧАЕМ
  • Список URL 10 конкурентов — сохраняем в Google Docs
  • Текст с болями аудитории — сохраняем целиком
  • Оба материала готовы для загрузки в NotebookLM

BAR-АНАЛИЗ КОНКУРЕНТОВ

⏱ 15 МИН
ШАГ 2 • NOTEBOOKLM

Загружаем сайты конкурентов и боли аудитории. AI читает за нас и вытаскивает неочевидные потребности. Это ваши конкурентные преимущества.

⚠ NotebookLM работает без ограничений. Сайты конкурентов — как Website-источники. Боли из Perplexity — как Copied Text.
  • Откройте notebooklm.google.com — New Notebook
  • Add Source — Website — вставьте URL конкурентов. Повторите для 5–7 сайтов
  • Add Source — Copied Text — вставьте весь текст болей из Perplexity
  • Ждите загрузку 1–2 минуты — вставляйте промпт в чат
ПРОМПТ №3 – BAR-АНАЛИЗ
Проанализируй все загруженные сайты конкурентов и блоки боли аудитории. Сделай BAR-анализ по структуре: 1. B — Что ДЕЛАЕТ ВСЕ — что обязательно есть почти на каждом сайте 2. A — Что НЕ ЗАКРЫВАЕТ НИКТО — боли клиентов, которые не закрывает ни один конкурент (по возможностям) 3. R — СИЛЬНЫЕ МЕСТА конкретных конкурентов — в чем они реально выделяются 4. На основе этих данных сформулируй список из 10–15 гипотез, какие новые офферы и формулировки мы можем использовать на нашем сайте, чтобы быть сильнее всех.
ПРОМПТ №4 – СТРУКТУРА ЛЕНДИНГА
На основе BAR-анализа предложи структуру лендинга для [ВАШ НИША]. Для каждого блока укажи: – Название секции – Главную задачу блока – Какую боль / желание закрывает – Какой основной оффер нужно озвучить – Ключевые сообщения для клиента Сделай таблицей в текстовом виде.
✓ ЧТО ПОЛУЧАЕМ
  • BAR-анализ — неожиданные потребности и ваши преимущества
  • Структуру лендинга с обоснованием каждого блока
  • Сохраняем в Google Docs — будет базой для текстов

ТЕКСТЫ НА ЯЗЫКЕ КЛИЕНТА

⏱ 15 МИН
ШАГ 3 • GEMINI

Пишем тексты всех секций на основе данных из NotebookLM. Не шаблонные фразы — конкретные ответы на страхи клиента.

⚠ Нужен Google-аккаунт и регион не РФ. Альтернатива: Cursor или Claude Code с тем же промптом — результат будет похожим.
  • Откройте gemini.google.com — выберите Gemini 2.5 Pro
  • Вставьте промпт №5 — BAR-анализ из NotebookLM в отмеченное место
  • Вставьте промпт №6 — тексты всех блоков и секций для лендинга
  • Скопируйте тексты и сохраните их в один документ
ПРОМПТ №5 – СИСТЕМНЫЙ КОНТЕКСТ
Ты — копирайтер для продуктово-сервисного рынка. Твоя задача — написать тексты лендинга, которые: – опираются на реальный спрос и боли клиентов, – учитывают анализ конкурентов, – показывают конкретную выгоду, а не общие слова, – говорят с клиентом простым человеческим языком, – каждый абзац отвечает на один конкретный вопрос клиента. Ниже — BAR-анализ и структура лендинга: [ВСТАВЬ BAR-АНАЛИЗ И СТРУКТУРУ ЛЕНДИНГА ИЗ NOTEBOOKLM]
ПРОМПТ №6 – ВСЕ ТЕКСТЫ ЛЕНДИНГА
Напиши тексты для всех секций лендинга. 1. HERO – Н1 (без громких слов, до 30 слов), подзаголовок, смарт-оффер, 3–5 буллетов, CTA (кнопка). 2. БЛОК О НАС – факты и цифры, без воды. 3. УСЛУГИ / ПРОДУКТ – что именно делаем, в чем результат. 4. КЕЙСЫ – 3–5 штук с цифрами до / после и результатом. 5. МАСТЕР / ЭКСПЕРТ – кто будет делать работу, кратко. 6. ВОПРОС–ОТВЕТ – 5–7 частых вопросов и ответы. 7. ФИНАЛЬНЫЙ БЛОК – повтор оффера и сильный призыв к действию. Язык: русский. Тон: живой, уверенный, без нагнетания и без канцелярита.
✓ ЧТО ПОЛУЧАЕМ
  • Тексты всех секций — как живые блоки, не шаблоны
  • Каждый абзац отвечает конкретной боли или страху
  • Сохраняем в Google Docs — для Antigravity

ДИЗАЙН И КОД – АВТОНОМНО

⏱ 15 МИН
ШАГ 4 • ANTIGRAVITY + STITCH

Один промпт — Antigravity автономно создаёт дизайн в Stitch, пишет React-код, собирает сайт. Вы ничего не настраиваете — AI работает в упряжке сам.

⚠ Нужен Google-аккаунт и регион не РФ. Альтернатива: Cursor или Claude Code с тем же промптом — результат будет похожим.
  • Откройте antigravity.google.com — New Project → терминал
  • Вставьте команду установки skill — Enter — 30 секунд
  • Вставьте главный промпт — замените переменные — ждите 7–10 минут
  • Stitch экспортирует сайт в отдельной вкладке — там рисуется дизайн. Не трогайте ничего.
КОМАНДА УСТАНОВКИ SKILL
npx skills add google-labs-code/stitch-skills --skill stitch-loop --global
ПРОМПТ №7 – ГЛАВНЫЙ ПРОМПТ ANTIGRAVITY
Using stitch-loop skill, create a complete landing page for [НАЗВАНИЕ БРЕНДА]. Аудитория: [АУДИТОРИЯ И ИХ ГЛАВНЫЕ СТРАХИ]. Design direction: – Style: [минимализм / контраст / монохромность] – Branding: [ЛОГОТИП / ЦВЕТА] – Accents: [АКЦЕНТНЫЙ ЦВЕТ] – Typography: крупные serif заголовки, clean sans serif body – Clean layout, много воздуха Content language: Russian Generate these page sections one by one: 1. Hero – H1, подзаголовок, CTA, «почему сейчас», 3–5 буллетов. 2. Services – 3–5 услуги 3. Team – 3 профиля специалистов 4. Testimonials – 3 аутентичных отзывов 5. Booking/Contact form Export to React with Tailwind CSS. Mobile responsive.
✓ ЧТО ПОЛУЧАЕМ
  • Универсальный дизайн всего лендинга в Stitch
  • React-код с Tailwind CSS — готов к публикации
  • Мобильная версия — адаптирована автоматически

ПУБЛИКАЦИЯ ЗА 3 КЛИКА

⏱ 5 МИН
ШАГ 5 • VERCEL

Публикуем сайт в интернет. Если у вас уже есть репозиторий — используйте вкладку «Вручную».

Вариант 1 — AI агент (авто). Antigravity сам создаст репозиторий и зальёт код, Vercel подхватит его автоматически.
  • В чате Antigravity напишите команду ниже — AI создаст GitHub-репозиторий сам
  • Откройте vercel.com — войдите через GitHub — New Project
  • Выберите репозиторий — Deploy — 60 секунд
  • Сайт живой — Vercel даст публичный URL
КОМАНДА ДЛЯ AI-АГЕНТА
Push this project to a new GitHub repository called [awesome-landing]. Then provide me the repository URL.
Вариант 2 — ВРУЧНУЮ (ТЕРМИНАЛ). Если вы хотите контролировать процесс публикации.
  • Локально инициализируйте git-репозиторий и закоммитьте код
  • Создайте пустой репозиторий на GitHub
  • Привяжите удалённый origin и выполните git push
  • Подключите репозиторий к Vercel и нажмите Deploy
КОМАНДЫ ПУБЛИКАЦИИ ВРУЧНУЮ
git init git add . git commit -m "first commit" git branch -M main git remote add origin [ВАШ_GITHUB_URL] git push -u origin main
✓ ГОТОВО
  • Сайт живой в интернете по публичному URL
  • SSL-сертификат (https) — автоматически
  • Можно привязать собственный домен
  • Правки — через git push или новый запуск Antigravity
